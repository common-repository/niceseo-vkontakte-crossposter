== Description ==

Plugin for Auto crossposting your new published Wordpress posts to wall on your "public page" on VK.com social network.

With Image, image posted to official public's album and attachment to wall post.

With Tags from WP post to hashtages on vk wall post

Link to Site posrt Attached to VK wall too.

by linur, blogger, SEO blog www.NiceSEO.ru

http://www.niceseo.ru/niceseo-s-vkontakte-crossposter

= Features =

* Auto post to your VK public on 0.001 seconds
* Image posted to official public's album and attachment to wall post
* Tags from WP post to hashtages on vk wall post
* Link to Site post Attached to VK wall too.
* Very good usability on admin panel
* Optimised! Weight of plugin = ~3kb!

== Installation ==

1. Install plugin.
2. Activate the plugin on the plugin screen.
3. Changed parametres on Parameters -> NiceSEO VKontakte Crossposter Settings
